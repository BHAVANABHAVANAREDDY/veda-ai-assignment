# VedaAI – AI Assessment Creator

Full Stack Engineering Assignment submission.

## Tech Stack
- Next.js
- React
- Node.js
- Express
- MongoDB
- Gemini API
- Vercel

## Features
- Create assignment
- Generate AI question paper
- Question sections
- Difficulty levels
- Marks allocation
- Responsive UI
- MongoDB storage
